第一次建站

听讲座会议的时候，看大佬都会有一个自己的网站，将一些成果和介绍放进去，感觉挺酷的

但是我：

  • 没有前端代码基础
  • 不想花钱租服务器

于是我选择了 github + hexo 的方式

我还花了9块钱在阿里云买了一年的域名 wuaodi.cn

经历了一天的折腾，填好了几个 坑，终于有了自己的一个小破站

想一想，这片土地上叫 wuaodi 的人应该挺多的，拿这个做域名的却只有我一个，酷酷的。

成长中成长

为了建网页，我又熟悉了 git 的一些流程

为了发博文，第一次使用了 markdown 编辑器（坚果云自带的）

原来也知道 git 和 markdown 是肯定要用的，学起来却觉得枯燥，现在是在一个具体目标的驱使下顺带学了，这就是在成长中成长——用什么学什么

目标驱使，在很多时候都更有效率，更有方向感吧

Go Go Go

即将正式开始研究生生涯了
